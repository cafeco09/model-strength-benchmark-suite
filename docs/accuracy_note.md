# Accuracy Note

The comparison method is valid for answering:

> Which model appears strongest for which task category?

It is not valid for answering:

> Which model is objectively the best model?

To make the method more accurate:

1. Run all models on all tracks.
2. Keep prompt settings identical where possible.
3. Record model IDs and run dates.
4. Use blind human rating for emotional and creative tasks.
5. Use unit tests or executable checks for coding tasks.
6. Include cost and latency.
7. Include safety and sycophancy penalties.
8. Separate public benchmark references from your own experiment results.
